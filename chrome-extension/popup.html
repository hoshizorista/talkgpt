TALK-TO-ChatGPT

To start using this extension, you need to open the ChatGPT home page. Once logged in, a box will appear at the top right with a 'START' button. Click START to begin.

This extension allows you to talk to ChatGPT using your microphone with the browser's voice recognition API. When the bot responds, the reply is read aloud with a voice of your choice.

  • When the box's bottom border is red, the microphone is listening.
  • When the box's bottom border is green, the bot is talking.
  • Access settings by clicking the gear icon to change voice/language, speed, and more.

Check GitHub Repo for Troubleshooting Feel free to contribute to the project!

Having trouble getting it to work?

Issues with the extension? Check the project page for the latest updates. OpenAI updates can sometimes break the extension, but I'll fix it quickly.

  • This extension is designed for Chromium-based desktop browsers. If it doesn’t work in your browser, it's because the necessary APIs aren't supported.
  • If voice recognition doesn’t work, check this test page.
  • If the bot's voice is scrambled, change the voice settings or restart the browser to reset Chromium’s voice API bug.

If your issue isn't listed, visit the GitHub Issues tab for more support.
